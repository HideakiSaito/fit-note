# Awesome-name
トレーニング管理アプリ

## Description
トレーニングの重量、セット数、メモを記録。前回のトレーニングからテンプレートを作成可能。
トレーニングの推移をチャートで確認。データのバックアップ、リカバリ対応（エクスポート、インポート）

![Demo](https://image-url.gif)

## Features
- トレーニングの重量、セット数、メモを記録。
- 前回のトレーニングからテンプレートを作成可能。
- トレーニングの推移をチャートで確認。
- データのバックアップ、リカバリ対応（エクスポート、インポート）
- トレーニング場所ごとの検索機能。 
- 種目の追加、変更。
- 部位、トレーニングメソッド毎の管理。
